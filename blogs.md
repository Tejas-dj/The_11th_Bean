## How to add a post

Copy the template block below, paste it into `data/blog.ts` inside the `BLOG_POSTS` array, and fill in the fields.
All fields are required except `body` paragraphs beyond the first one.

**Category options:** Behind the Brew · Craft & Process · Coffee 101 · Origin Story · People · Notes

```
{
  slug: "url-friendly-title-with-dashes",        // used in the URL: /blog/slug
  title: "Full Post Title Here",
  date: "YYYY-MM-DD",                            // e.g. 2026-05-01
  category: "Behind the Brew",
  readingTime: 4,                                // estimated minutes
  excerpt: "One or two sentences. This shows on the blog listing card.",
  body: [
    "First paragraph goes here.",
    "Second paragraph goes here.",
    "Each string in this array is its own paragraph.",
  ],
},
```
